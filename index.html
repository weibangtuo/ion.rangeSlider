Ion.RangeSlider Demo

Learn how to use Ion.RangeSlider on several practical demos

Simple start, basic params

Set min value, max value and start point


      new IonRangeSlider(document.getElementById('demo_0'),  {
          min: 100,
          max: 1000,
          from: 550
      });

Set type to double, specify range, show grid and add a prefix "$"


      new IonRangeSlider(document.getElementById('demo_1'),  {
          type: 'double',
          grid: true,
          min: 0,
          max: 1000,
          from: 200,
          to: 800,
          prefix: '$'
      });

Using array of custom values

Set up you own numbers


      var custom_values = [0, 10, 100, 1000, 10000, 100000, 1000000];
      
      // be careful! FROM and TO should be index of values array
      var my_from = custom_values.indexOf(10);
      var my_to = custom_values.indexOf(10000);
      
      new IonRangeSlider(document.getElementById('demo_5'),  {
          type: 'double',
          grid: true,
          from: my_from,
          to: my_to,
          values: custom_values
      });

Values array could be anything, even strings


      new IonRangeSlider(document.getElementById('demo_6'),  {
          grid: true,
          from: new Date().getMonth(),
          values: [
              "Jan", "Feb", "Mar", "Apr", "May", "Jun",
              "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"
          ]
      });

Change visual look of numbers (prettify)

Improve readability of big numbers, like 10000000000 ðŸ˜„


      new IonRangeSlider(document.getElementById('demo_7'),  {
          skin: 'big',
          grid: true,
          min: 1000,
          max: 1000000,
          from: 100000,
          step: 1000,

Use your own prettify function to transform numbers whatever you like. For example calculating logarithm of a number


      function my_prettify (n) {
          var num = Math.log2(n);
          return n + " → " + (+num.toFixed(3));
      }
      
      new IonRangeSlider(document.getElementById('demo_8'),  {
          skin: 'big',
          grid: true,
          min: 1,
          max: 1000,
          from: 100,
          prettify: my_prettify
      });

Manipulating UI

Disable enable any UI elements


      new IonRangeSlider(document.getElementById('demo_12'),  {
          skin: 'big',
          type: 'double',
          min: -100000,
          max: 100000,
          from: -100000,
          to: 100000,
          step: 10000,
          grid: true,             // show/hide grid
          force_edges: false,     // force UI in the box
          hide_min_max: false,    // show/hide MIN and MAX labels
          hide_from_to: false,    // show/hide FROM and TO labels
          block: false            // block instance from changing
      });

Customizing Grid

Manipulate the grid using build in parameters


    new IonRangeSlider(document.getElementById('demo_0'),  {
        min: 0,
        max: 10000,
        from: 777,
        step: 1,            // default 1 (set step)
        grid: true,         // default false (enable grid)
        grid_num: 4,        // default 4 (set number of grid cells)
        grid_snap: false    // default false (snap grid to step)
    });

Lock and restrict handles

Lock handles to prevent moving


    new IonRangeSlider(document.getElementById('demo_14'),  {
        type: 'double',
        min: 0,
        max: 10,
        from: 2,
        to: 8,
        grid: true,
        grid_snap: true,
        from_fixed: false,  // fix position of FROM handle
        to_fixed: false     // fix position of TO handle
    });

Limit movement of a handle


          new IonRangeSlider(document.getElementById('demo_15'),  {
              type: "single",
              min: 0,
              max: 1000,
              from: 500,
              grid: true,
              from_min: 250,      // set min position for FROM handle (replace FROM to TO to change handle)
              from_max: 750,      // set max position for FROM handle
              from_shadow: true   // highlight restriction for FROM handle
          });

Restrict interval size and/or drag it

              to: 600,

          });

Working with dates

Restrict interval size and/or drag it


          var lang = "en-US";
          var year = 2018;
          
          function dateToTS (date) {
              return date.valueOf();
          }
          
          function tsToDate (ts) {
              var d = new Date(ts);
          
              return d.toLocaleDateString(lang, {
                  year: 'numeric',
                  month: 'long',
                  day: 'numeric'
              });
          }
          
          new IonRangeSlider(document.getElementById('demo_17'),  {
              skin: 'big',
              type: 'double',
              grid: true,
              min: dateToTS(new Date(year, 10, 1)),
              max: dateToTS(new Date(year, 11, 1)),
              from: dateToTS(new Date(year, 10, 8)),
              to: dateToTS(new Date(year, 10, 23)),
              prettify: tsToDate
          });
